🏗️ Backend FastAPI del Equipo
  • Modelos SQLModel: Insured, Policy, Vehicle, Incident, Claim, Case
  • Dataset real: 1000+ registros de reclamos con campo `fraud_reported`
  • APIs REST: Endpoints completos para gestión de seguros
  • Detección de fraudes: Sistema base con análisis de patrones
  • Base de datos: SQLite con datos reales de seguros
🎨 Frontend Bootstrap del Equipo
  • index.html: Dashboard principal con métricas y Chart.js
  • policies.html: Gestión de pólizas con filtros y paginación
  • claims.html: Gestión de reclamos y seguimiento
  • Sistema de temas claro/oscuro con localStorage
  • APIs: /policies, /claims, /insureds, /stats
🎯 Módulo Analytics Especializado:

Módulo de analytics avanzado que complementa perfectamente el sistema de seguros del equipo:

  • Utiliza el dataset de fraudes: Analiza los 1000+ registros con `fraud_reported`
  • Consume APIs del equipo: /policies, /claims, /stats del backend colaborativo
  • Análisis especializado: Métricas de riesgo y detección avanzada de fraudes
  • Integración limpia: Módulo independiente que no interfiere con el trabajo del equipo
